fl_webbridge_tool Architecture

Flutter App Shell · Bridge Plugin · H5 WebView · Capability Handlers · Full-Chain Observability

Bridge Protocol

  • · JS: window.flutter_inappwebview.callHandler('BR_WebNativeBridge', msg)
  • · Flutter: BRWebBridge.bind(controller) → addJavaScriptHandler
  • · Msg: { id, action, params } → response { id, ok, data }
  • · Lifecycle: emitLifecycle() → container.lifecycle

Capabilities (16+ actions)

  • · 📷 takePhoto / 🎥 takeVideo / 🎙️ startRecord
  • · 📂 pickFile / 👁️ preview / 🗑️ delete
  • · 🧭 navigate / ✏️ setTitle / 🎨 hideTabBar
  • · 📶 network.status / 📱 system.info / 📦 resource.*

Observability

  • · CallbackBRWebLogger → onLog(BRWebLogEntry)
  • · 📡Lifecycle ⬆️Request ⬇️Response 📜Console 💥Error 🎨UI
  • · BRWebNetworkMonitor → real-time net status
  • · BRWebSystemInfo → device/os/version dumps
