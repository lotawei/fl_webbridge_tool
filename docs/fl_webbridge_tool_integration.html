1. H5 → 原生 API 参考

所有 action 通过 window.BR_WebContainer.call(action, params) 调用，返回 Promise。

1.1 📷 拍照

await BR_WebContainer.call('device.camera.takePhoto', {
  maxWidth:   1600,        // 可选，最大宽度 px
  maxHeight:  1600,        // 可选，最大高度 px
  maxSizeKB:  1024,        // 可选，最大文件大小 KB，默认 1024（1MB）
  saveToGallery: true,     // 可选，默认 true → 存入系统相册
});

返回：

{
  cancelled: false,
  path:      "/tmp/photo_xxx.jpg",
  name:      "photo_xxx.jpg",
  mimeType:  "image/jpeg",
  size:      345678,         // 字节数
  sizeKB:    337,            // KB
  savedToGallery: true,
  galleryPath: "file:///..."  // iOS 相册路径
}

1.2 🖼️ 从相册选照片 新增

await BR_WebContainer.call('device.camera.pickPhoto', {
  maxWidth:   1600,
  maxSizeKB:  1024,        // 默认 1MB
});

返回格式同 1.1（无 savedToGallery 字段）。

压缩策略： 默认 ≤1MB。如果原图超限，自动用 quality:85 → 60 → 40 → 20 → 10 逐级压降，直到满足上限。要更大上限传 maxSizeKB: 102400，要不压缩传极大值。

1.3 🎥 录像

await BR_WebContainer.call('device.camera.takeVideo', {
  maxDuration:  15,              // 最大时长 秒，默认 30
  camera:       'rear',          // 'rear' | 'front'，默认 rear
  saveToGallery: true,           // 默认 true
});

1.4 🎬 从相册选视频

await BR_WebContainer.call('device.camera.pickVideo', {
  maxDuration: 600,             // 默认 600 秒
});

1.5 📂 选择文件

await BR_WebContainer.call('device.file.pick', {
  multiple: true,               // 是否多选
});

返回：

{
  cancelled: false,
  files: [{ name: "xxx.pdf", path: "/tmp/xxx", size: 123456, extension: "pdf" }]
}

1.6 👁️ 预览文件

await BR_WebContainer.call('device.file.preview', {
  path:      "/tmp/photo.jpg",
  type:      "image",          // "image" | "video" | "audio"
  title:     "预览",
  mimeType:  "image/jpeg",    // 可选
});

1.7 🗑️ 删除文件

await BR_WebContainer.call('device.file.delete', {
  path: "/tmp/photo.jpg",
});

1.8 🎙️ 录音

// 开始录音
await BR_WebContainer.call('device.audio.startRecord');
// → { recording: true, path: "/path/to/xxx.m4a" }

// 停止录音
await BR_WebContainer.call('device.audio.stopRecord');
// → { recording: false, path: "/path/to/xxx.m4a" }
录音文件存 documents 目录，不会被系统 temp 清理。

1.9 🧭 导航控制

// 跳转新页面（需先在 Flutter 侧注册路由）
await BR_WebContainer.call('navigation.navigateTo', {
  route:  '/h2',
  params: { id: 123 },        // 可选，传给目标页
});

// 返回上一页
await BR_WebContainer.call('navigation.goBack');

// 设置页面标题
await BR_WebContainer.call('navigation.setTitle', {
  title: '新标题',
});

1.10 🎨 UI 控制

await BR_WebContainer.call('ui.hideTabBar');
await BR_WebContainer.call('ui.showTabBar');

1.11 🚪 关闭容器

await BR_WebContainer.call('container.close', {
  reason: 'h5_request',
});

2. 原生 → H5 事件（H5 接收）

Flutter 侧通过 BRWebBridge.callWeb(method, params) 推送到 H5，在 window.BR_WebContainer.__nativeCall(payload) 中接收。

2.1 App 生命周期 新增

事件payload.methodpayload.params
App 回到前台app.lifecycle{ state: 'foreground', timestamp }
App 进入后台app.lifecycle{ state: 'background', timestamp }
App 非活跃app.lifecycle{ state: 'inactive', timestamp }
容器销毁app.lifecycle{ state: 'disposed', timestamp }
页面可见page.visibility{ visible: true, timestamp }
页面被覆盖page.visibility{ visible: false, timestamp }

2.2 WebView 生命周期

事件payload.methodpayload.params
加载开始container.lifecycle{ type: 'loadStart', url }
加载完成container.lifecycle{ type: 'loadStop', url }

2.3 Flutter 主动调用

// Flutter → H5
bridge.callWeb('native.ping', { time: DateTime.now().toIso8601String() });

// 任意自定义事件
bridge.callWeb('payment.success', { orderId: 'xxx', amount: 99.00 });

H5 侧接收（链式处理，多个 composable 共存）：

// 在 useBridge / useAppLifecycle 中已自动挂载
window.BR_WebContainer.__nativeCall = (payload) => {
  // payload = { method: 'app.lifecycle', params: { state: 'foreground' } }
  console.log('收到原生调用', payload);
  return { received: true };
}

4. App 生命周期感知（H5 侧） 新增

import { useAppLifecycle } from './composables/useAppLifecycle'

const { appState, pageVisible, label } = useAppLifecycle()

// appState: ref<'foreground' | 'background' | 'inactive' | ...'>
// pageVisible: ref
// label(): '🟢 foreground · 可见'

// 监听变化
watch(appState, (state) => {
  if (state === 'background') pauseMedia()
  if (state === 'foreground') resumeMedia()
})

模板中直接显示：{{ label() }} → 🟢 foreground · 可见

5. 路由导航

5.1 注册路由

// 注册 WebView 路由
BRWebNavigator.register('/h1', BRWebRouteConfig(
  url: 'assets/h5/demo.html', title: 'H1 首页',
));
BRWebNavigator.register('/vue', BRWebRouteConfig(
  url: 'http://172.16.2.158:5173',
  title: 'Vue3 演示',
  initialData: BRWebInitialData(
    accessToken: 'xxx',
    userData: {'id': '1001', 'name': 'lotawei'},
    lang: 'zh',
  ),
));

// 注册原生页面路由
BRWebNavigator.registerNative('/settings', (ctx, params) => SettingsPage());

5.2 跳转

// Flutter 侧
BRWebNavigator.push(context, '/h2', params: { id: 123 });
BRWebNavigator.pop(context);

// H5 侧
await BR_WebContainer.call('navigation.navigateTo', { route: '/h2' });
await BR_WebContainer.call('navigation.goBack');
注意： BRWebNavigator.push() 创建的页面会自动注入全局日志适配器，无需手动传 logger。

7. 扩展业务能力

class AppCapabilityHandler implements BRWebCapabilityHandler {
  @override
  Future<Object?> handle(BuildContext context, BRWebBridgeMessage message) async {
    return switch (message.action) {
      'user.getToken'    => {'token': 'your-token'},
      'payment.create'   => _createPayment(message.params),
      'analytics.track'  => _trackEvent(message.params),
      _ => BRWebCapabilityHandlerResult.notHandled,  // 交给下一个 handler
    };
  }
}

// 链式注册：自定义 → 默认（拍照/录音等）
BRWebContainerPage(
  capabilityHandler: CompositeBRWebCapabilityHandler([
    AppCapabilityHandler(),
    DefaultBRWebCapabilityHandler(),
  ]),
);

8. Vue3 前端封装

8.1 useBridge — Bridge 通信

import { useBridge } from '../composables/useBridge'

const { logs, bridgeReady, isInWebView, call, appendLog } = useBridge()

// 调用原生能力
const result = await call('device.camera.takePhoto', { maxWidth: 1600 })
appendLog(result)  // 同步记录到本地日志面板

浏览器模式自动降级为 mock 返回，方便本地开发。

8.2 useBRData — 注入数据 新增

import { useBRData } from '../composables/useBRData'

const { brData, isLoggedIn, userName, lang } = useBRData()

isLoggedIn()  // → true | false
userName()    // → "lotawei"
lang()        // → "zh"
brData.value  // → { accessToken, user: {id,name}, lang, ... }

数据由 Flutter 侧通过 initialUserScripts + AT_DOCUMENT_START 注入为 window.__BR_Data__，H5 无需 bridge 即可同步读取。

8.3 useAppLifecycle — 生命周期 新增

import { useAppLifecycle } from '../composables/useAppLifecycle'

const { appState, pageVisible, label } = useAppLifecycle()

watch(appState, s => s === 'background' ? pause() : resume())

10. 性能与最佳实践

  • Tab 场景用 IndexedStack 缓存 WebView，避免频繁销毁重建。
  • 大文件不通过 Bridge 传 base64，传 path / id。
  • 拍照默认 ≤1MB（maxSizeKB: 1024），可调。
  • Bridge 请求自动走全局日志，无需手动记录。
  • SPA 页面关注 history 变化，不能只依赖 onLoadStop。
